# final 키워드
    변수, 메서드가 변경불가능하도록 만든다

    클래스 적용시
    1. 확장불가능
    2. 변경불가능
    3. 다른 클래스의 조상 클래스 불가능

    메서드 적용시
    1. 변경불가능
    2. 오버라이딩 불가능

    변수 적용시
    1. 변경불가능

# finally 키워드
    try-catch 블록에서 에러가 발생하더라도 항상 실행될 코드를 작성하기 위해 사용된다.

# finalize 메서드
    GC(Garbage Collector)가 더 이상 참조되지 않는 동적 객체를 Heap 영역에서 삭제하기 위해 호출하는 메서드이다.
